# StoreTestApp_Android

[![Android CI](https://github.com/hoganyang/StoreTestApp_Android/actions/workflows/android_ci.yml/badge.svg)](https://github.com/hoganyang/StoreTestApp_Android/actions/workflows/android_ci.yml)

A modern Android application built with **Kotlin** and **Jetpack Compose**.

## Tech Stack
*   **Language**: Kotlin
*   **UI**: Jetpack Compose (Material 3)
*   **Build**: Gradle Kotlin DSL
*   **CI**: GitHub Actions

## Getting Started

1.  **Clone**: `git clone https://github.com/hoganyang/StoreTestApp_Android.git`
2.  **Open**: Import into Android Studio (JDK 17+).
3.  **Run**: Press ▶️ to build and deploy.

> **Note**: This project is pre-configured with **Aliyun** (dependencies) and **Tencent** (Gradle) mirrors for fast builds in China.
