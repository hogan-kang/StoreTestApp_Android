【App Name / 应用名称】
StoreTestApp

【Short Description / 简短描述】(Max 80 chars)
A sample application demonstrating automated CI/CD deployment to Google Play.

【Full Description / 完整描述】(Max 4000 chars)
StoreTestApp is a demonstration project designed to showcase the complete workflow of modern Android development and deployment.

Key Features:
- Built with Kotlin and Jetpack Compose.
- Material 3 Design implementation.
- Automated Build & Signing using GitHub Actions.
- Continuous Deployment to Google Play Console.

This app serves as a reference implementation for developers looking to set up their own automated release pipelines.

1. App Icon (应用图标)
   - Size: 512 x 512 px

3. Phone Screenshots (手机截图)
   - Count: At least 2 screenshots
   - Ratio: 16:9 or 9:16 (Portrait)
